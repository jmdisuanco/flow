# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-01-03

### Features
* **Rebrand**: Renamed library package to `@jmdisuanco/flow`.
* **TypeScript**: Complete conversion to a TypeScript-first source codebase.
* **Biome**: Adopted Biome (v2) for linting and formatting.
* **Refactor**: Renamed core `flow` primitive to `pipe` for clarity.
* **Cleanup**: Removed legacy `GraphRunner` and complex graph topology features to focus on functional pipeline primitives.
